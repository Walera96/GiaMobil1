# Отчёт о нагрузочном тестировании системы ГИА

## Цель
Оценить производительность backend API под нагрузкой 50 виртуальных пользователей (членов ГЭК), голосующих одновременно.

## Инфраструктура теста
- **JMeter версия:** 5.6.3
- **Профиль нагрузки:** 50 VU, ramp-up 5 мин, длительность 15 мин
- **Сценарий:** login → цикл 10 итераций (GET /meetings + POST /votes с think time 500–1500 мс)

## Целевые метрики
| Метрика | Целевое значение | Фактическое (заполнить после прогона) |
|---------|------------------|---------------------------------------|
| Average Response Time | < 1000 мс | — |
| 95th Percentile | < 2000 мс | — |
| Error Rate | < 1% | — |
| Throughput | > 10 RPS | — |

## Запуск теста
```bash
cd backend/src/test/jmeter
jmeter -n -t gia-load-test.jmx -l results.jtl -e -o report/
```
